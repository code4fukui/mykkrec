# mykkrec - My Health Observation App

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple app to record daily health observations.

## Demo
https://code4fukui.github.io/mykkrec/

![mykkrec](mykkrec.png)
## Features
- Record daily health notes with timestamp
- Stores data in browser localStorage

## Usage
1. Open the app in a web browser
2. Type a note in the input field and click the "memo" button
3. The note, along with the current date and time, will be added to the list below
